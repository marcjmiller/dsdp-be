## Dependencies
- [Earthly](https://earthly.dev/get-earthly)
- [Docker](https://docs.docker.com/engine/install/)
- [Docker-compose](https://docs.docker.com/compose/install/)


## To get started with Development
- Clone the Repo
- Run `earthly +run-dev`

## Run tests
- Run `earthly +all`

## Run Lint and Formatting
- Run `earthly +lint`
- Run `earthly +format`
